  2. Add your slides using <section> elements with the class .slide.
  3. Style them using built-in classes like .slide-title or .slide-triad.

Key Classes

  • .slide: The base class for every slide.
  • .slide-title: A full-screen, centered title slide.
  • .slide-split: A two-column layout for comparisons.
  • .slide-triad: A three-column layout for balanced content.
  • .text-white: Makes your text white (great for dark backgrounds).
  • .fit: Ensures your images stay proportional and within bounds.
